匯入花醫班表 Excel

支援你現有的 Excel 格式，系統會自動解析。

已偵測到你的班表格式：
• 工作表名稱 = 月份（如「六月2」、「班表」）
• 第7列起為資料：院區 / 工作 / 姓名 / 班型 / 第1日...第31日
• 班別符號：日（白班 07-15）、小（小夜 15-23）、大（大夜 23-07）、休（休假）
• 空白 = 未排班（正常班週末）

Firebase 雲端同步設定

設定後，班表資料會自動同步到 Firebase，網頁版與手機 App 都可以即時看到最新班表。

1
前往 console.firebase.google.com，建立新專案（免費）
2
進入專案 → Build → Realtime Database → 建立資料庫（選 Asia-Southeast1）
3
Rules 設定為允許已登入使用者讀寫：
{"rules": {"schedule": {".read": "auth != null", ".write": "auth != null"}}}
4
Authentication → Sign-in method → 啟用 Google
5
專案設定（齒輪）→ 你的應用程式 → 選 Web → 複製 firebaseConfig 各欄位填入下方
