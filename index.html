Firebase 雲端同步設定

設定後，班表資料會自動同步到 Firebase，網頁版與手機 App 都可以即時看到最新班表。

1
前往 console.firebase.google.com，建立新專案（免費）
2
進入專案 → Build → Realtime Database → 建立資料庫（選 Asia-Southeast1）
3
Rules 設定為允許已登入使用者讀寫：
{"rules": { "schedule": {".read": "auth != null", ".write": "auth != null"}, "policy": {".read": "auth != null", ".write": "auth != null"}, "changeRequests": {".read": "auth != null", ".write": "auth != null"}, "leaveOrder": {".read": "auth != null", ".write": "auth != null"}, "userPrefs": {"$uid": {".read": "$uid === auth.uid", ".write": "$uid === auth.uid"}} }}
4
Authentication → Sign-in method → 啟用 Email/Password
5
專案設定（齒輪）→ 你的應用程式 → 選 Web → 複製 firebaseConfig 各欄位填入下方

帳號權限管理 僅管理員可見

設定各帳號層級：管理員（可改任何日期）・主任（可直接修改，可審核申請）・一般員工（超過期限需提申請）

修改期限設定

一般員工可自由修改班表的截止日：「班表月份的上個月幾號」。
例：設 25 號 → 3 月班表需在 2 月 25 日前完成修改，之後需申請審核
